# Global Styles Redesign: Frontend UI Redesign

**Feature**: Frontend UI Enforcement & Visual Redesign
**Design Document**: Global Styles Redesign
**Date**: 2026-01-23

## Color Palette

### Primary Colors
- **Primary**: #6366f1 (Indigo-500) - Main brand color
- **Primary Dark**: #4f46e5 (Indigo-600) - Hover states
- **Secondary**: #f3f4f6 (Gray-100) - Background accents
- **Accent**: #8b5cf6 (Violet-500) - Highlights and call-to-actions

### Semantic Colors
- **Success**: #10b981 (Emerald-500) - Completed tasks, success states
- **Warning**: #f59e0b (Amber-500) - Priority indicators
- **Error**: #ef4444 (Red-500) - Error states
- **Text**: #1f2937 (Gray-800) - Primary text
- **Text Muted**: #6b7280 (Gray-500) - Secondary text

### Background Colors
- **Background**: #ffffff (White) - Main background
- **Card Background**: #f9fafb (Gray-50) - Card containers
- **Surface**: #f3f4f6 (Gray-100) - Secondary surfaces

## Typography Scale

### Font Sizes
- **Heading XL**: 2.25rem (36px) - Main page titles
- **Heading LG**: 1.875rem (30px) - Section headings
- **Heading MD**: 1.5rem (24px) - Subsection headings
- **Heading SM**: 1.25rem (20px) - Small headings
- **Body LG**: 1.125rem (18px) - Large body text
- **Body MD**: 1rem (16px) - Standard body text
- **Body SM**: 0.875rem (14px) - Small text, labels
- **Caption**: 0.75rem (12px) - Helper text

### Font Weights
- **Bold**: 700 - Emphasis and headings
- **Medium**: 500 - Subtle emphasis
- **Regular**: 400 - Default text
- **Light**: 300 - Secondary text

### Line Heights
- **Tight**: 1.25 - Dense text blocks
- **Snug**: 1.375 - Body text
- **Normal**: 1.5 - Standard paragraphs
- **Relaxed**: 1.625 - Readable content

## Spacing Scale

### Spacing Units
- **XS**: 0.25rem (4px) - Micro adjustments
- **SM**: 0.5rem (8px) - Small gutters
- **MD**: 0.75rem (12px) - Standard spacing
- **LG**: 1.25rem (20px) - Moderate spacing
- **XL**: 1.75rem (28px) - Large separation
- **2XL**: 2.5rem (40px) - Major sections

## Border Radius

### Radius Scale
- **SM**: 0.25rem (4px) - Small elements
- **MD**: 0.375rem (6px) - Standard components
- **LG**: 0.5rem (8px) - Cards and containers
- **FULL**: 9999px - Circular elements

## Shadows

### Shadow Levels
- **SM**: 0 1px 2px 0 rgba(0, 0, 0, 0.05) - Subtle elevation
- **MD**: 0 4px 6px -1px rgba(0, 0, 0, 0.1) - Standard cards
- **LG**: 0 10px 15px -3px rgba(0, 0, 0, 0.1) - Elevated elements
- **XL**: 0 20px 25px -5px rgba(0, 0, 0, 0.1) - Prominent elements

## Global Component Classes

### Button Styles
- **Primary**: Indigo background, white text, indigo hover
- **Secondary**: White background, indigo text, gray hover
- **Outline**: Transparent background, indigo border/text, indigo hover
- **Ghost**: Transparent background, indigo text, indigo/10 hover

### Card Styles
- **Default**: White background, gray border, md padding, lg radius
- **Interactive**: Hover shadow, subtle transition
- **Highlighted**: Subtle primary border, slight elevation

### Input Styles
- **Default**: Gray border, white background, focus-indigo border
- **Error**: Red border, focus-red border
- **Success**: Green border, focus-green border
